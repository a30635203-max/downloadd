# YouTube Downloader App

تطبيق لتحميل الفيديوهات من اليوتيوب على نظام Android.

## الميزات:
- تصفح اليوتيوب داخل التطبيق
- تحميل الفيديوهات بجودات متعددة
- تحميل الصوت بصيغة MP3
- واجهة مستخدم جميلة مع دعم الوضع الليلي
- إدارة التحميلات والبحث فيها

## كيفية البناء:

### باستخدام GitHub Actions:
1. ادفع الكود إلى مستودع GitHub
2. انتقل إلى تبويب Actions
3. انتظر حتى يكتمل البناء
4. حمل ملف APK من قسم Artifacts

./gradlew assembleDebug
